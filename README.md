#Project Setup

##Setup react-native:

```
brew install node
brew install watchman
npm install -g react-native-cli
```

##Run code:

```
git clone https://github.com/JustinWei2010/ReactNativeTutorial.git
cd ReactNativeTutorial
npm install
react-native run-ios
react-native run-android
```
